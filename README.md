# The URS Team Database

This repository contains every registered team in the URS.

## Getting Started

The primary and easiest way to register your team is through our Discord bot, **Brightling**, using the `/teamdb submit` command. It provides a user-friendly form to submit your team's information directly from Discord.

👉 **[Read the Brightling Registration Guide](docs/GUIDE.md)**

## Manual Registration

While the bot is recommended, more technically-adept members can register manually by submitting a pull request. If you choose this route, you must follow the format in the `.template` folder.

If you are going to do this manually, please communicate this to the UmaOps team in the UmaOps category in our Discord.
