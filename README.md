# The URS Team Database

This repository contains every registered team in the URS and is stored here. This is usually automated by our in-house bot, Brightling, but more technically-adept member(s) of a URS-associated team can register through here.


## Wait do I need to do this manually?

No, actually! As mentioned before, we have a bot that does this automatically through a command, but if you *REALLY* want to, you must follow the format in `.template`. The template folder will contain everything that is considered a valid entry in the database.

If you are actually going to do this manually, please communicate this to the UmaOps team in the UmaOps category in our Discord.
